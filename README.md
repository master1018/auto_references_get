# auto_references_get
读取PDF格式的论文中所有的参考文献，并可以选择的进行下载。

### 运行方式

```shell
./run.sh
```

运行后根据前端提示即可，下载的论文文件的路径为./src/output/paper/。

### 有关项目其他

要想成功运行，需要挂梯子，或者自己修改项目中的爬取链接。

脚本使用的是Firefox浏览器，需要安装（当然也可以自己修改代码，使用其他浏览器兼容）
